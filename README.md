# 🚀 FitTrack Pro – Fitness Tracker Application

## 📱 Optimus Automate Internship – App Development Domain

FitTrack Pro is a modern Flutter-based Fitness Tracker Application developed as part of the **Optimus Automate Internship Program**.

The application enables users to monitor workouts, track calories burned, analyze performance metrics, manage fitness goals, calculate BMI, and securely store workout records using Firebase.

---

## 📌 Project Overview

FitTrack Pro provides a complete fitness tracking experience by allowing users to:

* Log daily workout activities
* Track calories burned
* Monitor workout duration
* View workout history
* Edit and delete workout records
* Analyze performance through charts and statistics
* Track fitness goals and progress
* Calculate Body Mass Index (BMI)
* Securely authenticate users with Firebase Authentication
* Store workout data in Firebase Firestore
